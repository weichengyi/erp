# 系统安装向导指南

> 首次访问系统时，会自动进入安装向导

---

## 🎯 安装流程

```
首次访问系统
    ↓
自动跳转到安装页面
    ↓
步骤 1：配置数据库连接
    ↓
步骤 2：创建管理员账户
    ↓
步骤 3：完成安装
    ↓
跳转到登录页面
```

---

## 📋 安装步骤详解

### 步骤 1：数据库配置

**需要填写的信息：**

| 字段 | 说明 | 示例 |
|------|------|------|
| **数据库主机** | MySQL 服务器地址 | `mysql` (Docker) 或 `127.0.0.1` (本地) |
| **端口** | MySQL 端口 | `3306` |
| **数据库名** | 数据库名称 | `erp_trade` |
| **用户名** | 数据库用户名 | `erp_user` |
| **密码** | 数据库密码 | 你设置的密码 |

**Docker 部署默认配置：**
```
数据库主机：mysql
端口：3306
数据库名：erp_trade
用户名：erp_user
密码：erp_password_2026
```

**测试连接：**
- 点击"测试连接"按钮
- 显示"连接成功"后才能继续下一步

---

### 步骤 2：创建管理员账户

**需要填写的信息：**

| 字段 | 说明 | 要求 |
|------|------|------|
| **用户名** | 管理员登录名 | 3-20 个字符 |
| **邮箱** | 管理员邮箱 | 有效邮箱格式 |
| **密码** | 登录密码 | 至少 6 位 |
| **确认密码** | 再次输入密码 | 必须与密码一致 |

**建议：**
- 使用强密码（字母 + 数字 + 符号）
- 记录好用户名和密码
- 邮箱用于找回密码（如实现该功能）

---

### 步骤 3：完成安装

**安装成功后显示：**
- ✅ 安装成功提示
- 管理员用户名
- 密码提示（隐藏显示）

**点击"前往登录"：**
- 跳转到登录页面
- 使用刚创建的管理员账户登录

## 🔧 技术实现

### 前端实现

**安装页面路径：** `/install`

**文件位置：**
```
frontend/src/views/Install.vue
```

**功能：**
- 三步安装向导
- 表单验证
- 数据库连接测试
- 实时状态显示

### 后端 API

**安装状态检查：**
```
GET /api/install/status
```

**响应示例：**
```json
{
  "installed": false,
  "userCount": 0
}
```

**测试数据库连接：**
```
POST /api/install/test-db
```

**请求体：**
```json
{
  "host": "mysql",
  "port": 3306,
  "database": "erp_trade",
  "username": "erp_user",
  "password": "erp_password_2026"
}
```

**执行安装：**
```
POST /api/install
```

**请求体：**
```json
{
  "database": {
    "host": "mysql",
    "port": 3306,
    "database": "erp_trade",
    "username": "erp_user",
    "password": "erp_password_2026"
  },
  "admin": {
    "username": "admin",
    "email": "admin@example.com",
    "password": "YourPassword123",
    "confirmPassword": "YourPassword123"
  }
}
```

## ❓ 常见问题

### 问题 1：无法访问安装页面

**解决：**
- 检查前端路由配置
- 确保 Install.vue 文件存在
- 清除浏览器缓存

### 问题 2：数据库连接失败

**解决：**
- 检查数据库是否运行
- 检查数据库地址和端口
- 检查用户名和密码
- 检查网络连接

### 问题 3：安装后无法登录

**解决：**
- 确认管理员用户名和密码
- 检查数据库是否有用户记录
- 查看后端日志

### 问题 4：重复安装

**解决：**
- 系统会自动检测并阻止
- 如需重新安装，清空数据库

## 🛠️ 重新安装

如需重新安装系统：

### 方式 1：清空数据库

```sql
-- 删除所有用户
DELETE FROM sys_user;

-- 重启应用
```

### 方式 2：删除数据卷（Docker）

```bash
# 停止容器
docker-compose down

# 删除数据卷
docker volume rm erp-trade_mysql_data

# 重新启动
docker-compose up -d
```
